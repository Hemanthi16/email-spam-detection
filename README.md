# 📧 Email Spam Detection System

## 🚀 Overview
Machine Learning project that classifies messages as Spam or Not Spam using NLP techniques.

## 🧠 Tech Stack
- Python
- Scikit-learn
- TF-IDF Vectorizer
- Naive Bayes
- Streamlit

## ⚙️ How to Run
```bash
python model.py
streamlit run app.py
